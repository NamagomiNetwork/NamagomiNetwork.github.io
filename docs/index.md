# NamagomiNetwork

## Namagomi-botについて
生ゴミ鯖用に色々便利機能をつけた bot

リポジトリは[こちらから](https://github.com/NamagomiNetwork/Namagomi-bot)

### コマンドについて

#### コマンドの表記方法について
- `<>`：省略不可能パラメータ
- `[]`：省略可能パラメータ
- `|`：「または」を意味し、パラメータの複数指定は不可能
- `...`：複数パラメータを指定可能

#### コマンド実行方法について
コマンドは、`設定したプリフィックス!コマンド名`により実行可能です。

実行例：`g!about`
